Inserting line breaks and page breaks

You can choose where to insert a line break or page break, if you want to change where TextEdit automatically inserted one.

Before you can insert page breaks, you must set the document to wrap to the page by choosing Format > “Wrap to Page.”

Making text wrap to window size or paper size

To insert a line break or page break:

■

To add a line break, choose Edit > Insert > Line Break.

■

To insert a page break, choose Edit > Insert > Page Break.

To remove a line break or page break that you inserted, press Delete.
