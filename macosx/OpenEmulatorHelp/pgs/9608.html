Opening documents created by other applications

TextEdit can open Microsoft Word format files, Open Document files, RTF files, Web Archive files, HTML files, and plain text files. Most word-processing applications can save a file as one of these types. TextEdit preserves much of the formatting in a Word format file, Open Document file, or RTF file when you view it in TextEdit. TextEdit may support some elements that other applications use, such as tables, but may not support others.

To open a document created by another application:

Choose File > Open.

Locate the file you want to open, and select it.

If there is a specific text encoding you wish to use, choose it from the Plain Text Encoding pop-up menu; otherwise, leave the setting at Automatic.

If you want to ignore formatting styles and elements in the document, select “Ignore rich text commands.”

Click Open.
