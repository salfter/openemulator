Creating a table within a table

In TextEdit, you can create a table inside an existing table cell. This “nested” table lets you split a single cell into additional rows and columns.

To create a table within a table:

Select a cell in an existing table.

Choose Format > Text > Table, and click Nest Table.

Set the number of rows and columns you want to use.

After you set up the nested table, you can select cells in that table and format the text alignment, borders, and background for them.
