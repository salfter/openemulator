Selecting styled text throughout a document

You can find and select all text that matches a particular style (for example, all bold words in Arial Narrow font). You can search in a selection of text or in the entire document.

To select styled text:

Make sure the ruler is displayed by selecting Format > Text > Show Ruler.

Choose Other from the Styles pop-up menu in the ruler.

Select either Document Styles or Favorite Styles.

Use the arrow buttons to cycle through available styles until you find the one you want, then click Select.

Select to search in either the entire document or in selected text.

Click Select.

Select to search one or more of the following characteristics of the style you chose: font family, font, style, or ruler.
