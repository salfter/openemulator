Automating TextEdit tasks with AppleScript

TextEdit includes AppleScript terms that let you automate tasks using scripts. You can use scripts to add text, copy information, set styles, or export information from a TextEdit document for use by another application.

To see the TextEdit terms and write scripts, you can use Script Editor, included with Mac OS X.

Open Script Editor, in the AppleScript folder in the Applications folder.

Choose File > Open Dictionary.

Select TextEdit, and click Open.

To learn more about AppleScript and how to use scripting terms, see AppleScript Help, or visit the AppleScript website.
