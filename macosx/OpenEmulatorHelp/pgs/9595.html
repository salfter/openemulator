About document formats

You can use TextEdit to view, edit, and save files in a variety of formats:

■

Plain Text (.txt)

■

Rich Text Format (.rtf)

■

Rich Text Format Directory (.rtfd)

■

HTML (.html)

■

Web Archive (.webarchive)

■

Open Document (.odt)

■

Microsoft Word 2007 (.docx)

■

Microsoft Word 97 (.doc)

■

Microsoft Word 2003 XML (.xml)

Files saved in formats other than plain text can include graphics, paragraph and font formatting, tables, and lists.

Most word processors can open RTF files. Microsoft Word can open the Word format files. TextEdit supports a number of Word features, including tables, but some other features may not be supported.

To choose a document’s format:

■

To save a file as plain text, select Format > Make Plain Text, and then choose File > Save.

■

To save a file in a format other than plain text, choose Format > Make Rich Text, and then choose File > Save As. Choose the file format from the pop-up menu.
