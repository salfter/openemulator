Automating TextEdit tasks with Automator

Automator allows you to automate tasks by linking several actions into a workflow. For example, you can use Automator to export text from TextEdit and merge it with Address Book contacts.

To see the TextEdit tasks you can automate with Automator:

Open Automator, in the Applications folder.

Open Automator

Select TextEdit in the Library list.

To learn more about Automator, see Automator Help.
