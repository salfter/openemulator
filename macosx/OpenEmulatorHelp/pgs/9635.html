Saving your documents

To save a document, choose File > Save. If the document has not been saved before, you will have to name it and specify a location to save it. To save a copy of a document, choose File > Save As and save the document using a different name.

You can also specify additional settings for saving documents in TextEdit preferences.

To set saving preferences:

Choose TextEdit > Preferences and click “Open and Save.”

TextEdit makes a backup file when it’s saving your work. This backup is normally deleted after its done saving. If you want to keep the backup file for your documents, deselect "Delete the automatic backup file."

TextEdit can save your documents to autosave files at a specified interval. If TextEdit unexpectedly quits, it opens those autosave files the next time you open TextEdit. To save autosave files, choose an interval from the “Autosave modified documents” pop-up folder.

TextEdit adds ".txt" to the end of filenames for plain text files. If you don't want to automatically add this extension, deselect "Add '.txt' to plain text files."

To reset all “Open and Save” preferences to their original settings, click Restore All Defaults.
