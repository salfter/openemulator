Opening an HTML document

TextEdit can display HTML documents as you’d see them in a browser (though images may not be available in some cases). If you want, you can also view the HTML code for the document and edit it.

To view and edit HTML code in a document:

Choose File > Open.

Select the “Ignore rich text commands” checkbox in the Open dialog.

Locate the document and click Open.

To make TextEdit always open HTML documents in code-editing mode, choose TextEdit > Preferences and click “Open and Save.” Under “When opening a file,” select “Ignore rich text commands in HTML files.”
