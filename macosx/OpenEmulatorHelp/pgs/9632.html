Setting preferences for new documents

You can use TextEdit preferences to specify initial settings for each new document you create.

Choose TextEdit > Preferences and click New Document.

Choose a format.

Rich text allows special formatting such as bold or special typefaces. Plain text is very simple and does not allow formatting.

Select "Wrap to Page" if you want to use document margins when you view or print a page.

Leave this unselected to let text wrap according to the size of the document window.

Set an initial window size.

You can always drag the window to change its size.

Choose default fonts for plain text and rich text documents.

Specify document properties for rich text documents. You can edit these and other properties for individual documents using the Format menu.

Select options to check spelling as you type or show a ruler in the document window.

To change everything back to the original settings, click Restore All Defaults.
