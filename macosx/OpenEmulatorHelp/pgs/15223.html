Formatting text using styles

You can use the Styles pop-up menu to apply or save text and paragraph formatting. Styles can include font type (such as Courier), style (such as italic), color, and size, as well as paragraph and line spacing.

The Styles pop-up menu includes several commonly used styles (for example, bold), and you can save your own text formatting as a favorite style, which then appears in the pop-up menu. You can also access other text styles already used in the document.

Only Rich Text Format (RTF) and Word Format documents can be formatted.

To format text with styles:

■

To apply a commonly used or favorite style, select some text and choose the style from the Styles pop-up menu.

■

To apply a style already used in the current document, select the text you want to change, then choose Other from the Styles pop-up menu and select Document Styles. Click the Forward or Back button to scroll through available styles until you find the one you want, then click Apply.
