If SimpleText can’t open a document that you saved in TextEdit

You can open SimpleText documents in TextEdit. However, if you save the document in TextEdit, it is automatically converted to an RTF file.

To make sure SimpleText can open the document, you can save it as a plain text file.

To save a document as a plain text file:

Choose Format > Make Plain Text and then save your document.

However, when you convert a document to plain text, all formatting (such as bold text) is removed.
