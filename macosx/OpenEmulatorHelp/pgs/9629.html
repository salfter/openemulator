Creating a PDF document from a TextEdit document

You can create a PDF document when you print a TextEdit document.

To create a PDF document:

Open the document and choose File > Print.

Select any settings you want to use, then choose “Save as PDF” from the PDF pop-up menu.

If you need to save a document as a PostScript file, choose “Save as PostScript” from the PDF pop-up menu.
