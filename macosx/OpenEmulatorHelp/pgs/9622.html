Copying and pasting styles

You can copy styled text and apply the style to selected text. Only the style of the selected text will change; the selected word or phrase remains the same.

To copy and paste a style:

Select the text whose style you want to copy.

Choose Format > Font > Copy Style.

Select the text you want to restyle.

Choose Format > Font > Paste Style.

You can also copy and paste styled text in a document and make sure it matches the style of text nearby. Select and copy the text, then choose Edit > “Paste and Match Style.” The pasted text loses its original style and takes on the style of surrounding text.
