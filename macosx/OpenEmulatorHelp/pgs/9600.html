Setting margins, paragraph indentation, and tab stops

You can set margins, paragraph indentation, and tab stops by moving icons in the document’s ruler.

To set any of these options, you’re document, you must make your document rich text by choosing Format > Make Rich Text. And if you want to change margins, you must set the document to wrap to the page by choosing Format > “Wrap to Page.”

To set margins, paragraph indentation, and tab stops:

Choose Format > Text > Show Ruler to make the ruler appear, and then move the icons in the ruler.

Ruler icons for tab stops, paragraph indentation, and margins
■

To set margins, drag the downward-facing triangles.

TextEdit won’t let you set the margins to be less than one inch on every side.

■

To set paragraph indentation, drag the horizontal rectangle.

■

To create a tab stop, drag one from the tab stop palette on the right side of the ruler.

■

To remove a tab stop, drag it off the ruler.
