Making text wrap to window size or paper size

By default, TextEdit wraps lines of text based on the width of the window, so when you widen the window, the lines get longer. When you print a document, TextEdit matches the length of lines in the printed document to what you see on the screen. If the window is very wide, TextEdit must reduce the font size to get all of the words on the page.

If you want TextEdit to wrap lines of text to the page, so lines don’t change size when you change the window’s size, you need to wrap to the page.

To wrap text based on paper width:

Choose Format > “Wrap to Page.”

A rectangle appears in the window indicating the margins of the document.

To set the paper size you want to use, choose File > Page Setup and select a paper size and orientation.

You can automatically hyphenate long words at the ends of lines to make the text wrapping appear more smooth by choosing Format > Allow Hyphenation.
