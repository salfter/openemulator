Saving a favorite text style

You can add your favorite text styles to the Styles pop-up menu for convenient document formatting.

To add a style to the Styles pop-up menu:

Choose Other from the Styles pop-up menu.

If the selected text does not have the style you want to add, use the Forward and back buttons to browse through the document’s styles until you find the one you want.

Click Add To Favorites.

Type a name for the style.

If you want to use the font type shown when you apply the style, select “Include the font as part of the style.”

To use the spacing and tabs in the selected text, select “Include the ruler as part of the style.”

To remove a favorite style from the list, choose Other from the Styles pop-up menu, select Favorite Styles, choose the style to remove from the pop-up menu, and then click Remove From Favorites. You cannot remove the Default style.
